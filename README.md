# Cloud-Based Bus Pass System

##  Project Overview

The **Cloud-Based Bus Pass System** is an online ticket booking application developed using Flask and deployed on the cloud. It provides users with a simple and reliable way to book bus tickets online while maintaining ticket records, unique ticket identification, and fixed pricing.

The system focuses on improving reliability, reducing ticket loss, preventing incorrect pricing, and providing a seamless booking experience.

---

## Features

- Online bus ticket booking
- Passenger details collection
- Unique ticket ID generation
- Booking confirmation page
- Database storage for bookings
- Fixed pricing based on bus type
- Prevents ticket data loss
- Cloud deployment support
- Reliable online access
